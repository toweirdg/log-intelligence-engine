# Log Intelligence Engine 🚀

AI-powered backend system to analyze logs, detect anomalies, generate insights, and automate operational decisions.

## Problem

Modern systems generate massive logs. Engineers struggle to detect issues quickly, leading to delayed incident response.

## Solution

This system ingests logs, processes them asynchronously, uses AI to generate insights, and triggers automated actions.

## Features

* Log ingestion API
* Async processing with Celery
* Basic anomaly detection
* AI-based log summarization
* Decision engine (rule-based)

## Tech Stack

* FastAPI
* PostgreSQL
* Redis + Celery
* OpenAI API
* Docker
